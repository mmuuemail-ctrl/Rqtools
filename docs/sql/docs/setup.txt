1. V Supabase spusť nejdřív:
- docs/sql/001_schema.sql
- potom docs/sql/002_functions.sql

2. V Supabase Storage vytvoř bucket:
- files

3. Bucket files nastav jako public.

4. Vytvoř soubor .env.local podle .env.local.example

5. Doplň:
- NEXT_PUBLIC_BASE_URL
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY
- STRIPE_SECRET_KEY
- STRIPE_WEBHOOK_SECRET

6. Lokální spuštění:
npm install
npm run dev

7. Lokální Stripe webhook:
stripe listen --forward-to localhost:3000/api/stripe/webhook

8. Secret z Stripe CLI vlož do:
STRIPE_WEBHOOK_SECRET

9. Na Vercelu nastav stejné env proměnné jako v .env.local

10. Tato verze app používá:
- 1 účet = 1 QR
- content_type = text / url / media
- kreditní body
- free views
- paid subscription pro URL/media a lepší ceny
